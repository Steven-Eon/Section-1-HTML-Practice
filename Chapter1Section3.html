Knowledge Check For Chapter 1, Section 3

1. How do you create a paragraph in HTML?

By using the <p> tag

2. How do you create a heading in HTML?

By using the <h1, h2, h3, h4, h5, h6> tag depending on the heirarchy of presentation.

3. How many different levels of headings are there and what is the difference between them?

There are 6 different heading tags each specifying order of significance and size of text size.

4. What element should you use to make text bold and important?

By using the <strong> tag, you can make the text bold and important.

5. What element should you use to make text italicized to add emphasis to it?

By using the <em> (emphasis) tag, you can make text italicized to add emphasis to the text.

6. What relationship does an element have with any nested element within it.

It is specified as a parent/child relationship where the element holding the other element is the parent and the latter is the child.

7.What relationship do two elements have if they are at the same level of nesting?

They share a sibling relationship where the elements at the same level of nesting are siblings to one another.

8. How do you create HTML comments?

You can use the <!-- and the --> tag to specify a comment and place the text in between the two tags. This will not be rendered on the browser but will be visible on code.
